# update-lecture

Update a lecture's content based on changed source materials.

## Steps
1. Identify which source documents changed (from issue or impact-check)
2. Read current lecture content
3. Draft updates following `templates/lecture-outline.md`
4. Apply create-roast-revise cycle
5. Update lecture in Google Docs via workspace-mcp
6. Update ontology relations in Oxigraph
7. Update `catalog/manifests/lectures.yaml`
