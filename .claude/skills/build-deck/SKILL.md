# build-deck

Build or rebuild a slide deck from lecture content.

## Steps
1. Read lecture content and outline
2. Follow `templates/slide-outline.md` structure
3. Gather relevant diagrams from `diagrams/`
4. Create/update Google Slides via workspace-mcp
5. Insert diagram images where referenced
6. Update `catalog/manifests/decks.yaml`
7. Link deck to lecture in Oxigraph
