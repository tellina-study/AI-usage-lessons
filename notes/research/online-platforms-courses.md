## 1. DeepLearning.AI — "AI for Everyone" (Andrew Ng)

**Платформа:** Coursera / DeepLearning.AI
**URL:** https://www.coursera.org/learn/ai-for-everyone
**Преподаватель:** Andrew Ng
**Длительность:** ~7 часов, 4 недели
**Уровень:** Начальный, нетехнический
**Целевая аудитория:** Бизнес-профессионалы, менеджеры, все, кто хочет понять AI

### Структура по неделям

**Неделя 1: Что такое AI? (~1 ч 49 мин)**
- Общее определение Machine Learning и связь с AI/Deep Learning
- Large Language Models — объяснение для начинающих
- Данные: сбор, типы данных (структурированные vs неструктурированные)
- Терминология AI: связь deep learning, machine learning, data science, AI
- Нейронные сети — интуитивное объяснение с диаграммами
- Что делает компанию "AI-компанией"
- Возможности и ограничения AI (пример: распознавание авто vs интерпретация жестов)
- Введение в Deep Learning (опциональные видео)

**Неделя 2: Создание AI-проектов (~1 ч 26 мин)**
- Workflow машинного обучения: сбор данных -> обучение модели -> развертывание
- Workflow data science: сбор данных -> анализ -> рекомендации к действию
- Различие между Data Science и Machine Learning
- Техническая осуществимость и бизнес-ценность AI
- Использование готовых AI-решений vs создание собственных
- Установка ожиданий по точности (performance expectations)

**Неделя 3: Внедрение AI в компании (~2 ч 05 мин)**
- Кейсы: умные колонки, самоуправляемые автомобили
- **AI Transformation Playbook** (5 шагов):
  1. Пилотный проект
  2. Создание внутренней AI-команды
  3. Широкое AI-обучение по отделам
  4. Развитие стратегического преимущества
  5. Коммуникация со стейкхолдерами

**Неделя 4: AI и общество (~1 ч 35 мин)**
- Реалистичный взгляд на AI
- Дискриминация и предвзятость (Bias)
- Adversarial-атаки на AI
- Злонамеренное использование AI
- AI и развивающиеся экономики
- AI и рабочие места
- Условный базовый доход (Conditional Basic Income)

### Практические задания
- Квиз после каждой недели (концептуальный, не технический)

### Уникальная педагогика
- "1-секундное правило" для оценки технической осуществимости AI-задачи
- AI Transformation Playbook — практический фреймворк для организаций
- Полностью нетехнический — никакого кода
- Множество аналогий из реального бизнеса

### Что можно позаимствовать
- Структуру "Что это -> Как строить проекты -> Как внедрять -> Влияние на общество"
- AI Transformation Playbook как готовый фреймворк
- Подход к объяснению AI без кода через аналогии
- "1-секундное правило" осуществимости

**Платформа:** Coursera / DeepLearning.AI
**URL:** https://www.coursera.org/learn/generative-ai-for-everyone
**Преподаватель:** Andrew Ng

**Уровень:** Начальный, нетехнический

### Структура по неделям

**Неделя 1: Введение в генеративный AI (~1 ч 44 мин)**
- Что такое генеративный AI и LLM
- Как работает supervised learning и его связь с генеративным AI
- Ограничения LLM
- Общие use cases: Writing (написание), Reading (анализ текста), Chatting (диалог)
- Применение prompt engineering
- Квиз 1: What is Generative AI?
- Квиз 2: Generative AI Applications

**Неделя 2: Проекты генеративного AI (~2 ч 22 мин)**
- Жизненный цикл проекта генеративного AI (от концепции до запуска)
- Продвинутые техники: RAG, Fine-tuning, RLHF
- Стоимость облачных LLM-решений
- **Практическое упражнение 1:** Prompting an LLM in code
- **Практическое упражнение 2:** Reputation monitoring system
- Квиз 1: Software Applications
- Квиз 2: Advanced technologies: Beyond prompting

**Неделя 3: Генеративный AI в бизнесе и обществе (~1 ч 40 мин)**
- Анализ рабочих процессов на предмет возможностей AI-автоматизации

- Ответственный AI: fairness, transparency, privacy, security
- Квиз 1: Generative AI and business
- Квиз 2: Generative AI and society

### Уникальная педагогика
- Фреймворк анализа задач (task analysis) для оценки потенциала автоматизации
- Замена "туманных интуиций об AI на точные ментальные модели"
- Hands-on: кодирование промптов (опционально)
- Быстрый курс — завершается за выходные

### Как объясняют LLM/трансформеры неспециалистам
- Через аналогии с supervised learning
- Фокус на том, что LLM умеет и не умеет делать
- Use cases: writing/reading/chatting — понятная типология

### Что можно позаимствовать
- Типологию use cases (writing, reading, chatting) как способ объяснения
- Фреймворк анализа задач для автоматизации
- Структуру "как работает -> как строить проекты -> влияние на бизнес и общество"
- Практические упражнения с промптингом

## 3. DeepLearning.AI + AWS — "Generative AI with Large Language Models"

**Платформа:** Coursera
**URL:** https://www.coursera.org/learn/generative-ai-with-llms
**Преподаватели:** Andrew Ng (вступления), инструкторы AWS и DeepLearning.AI
**Длительность:** ~16 часов, 3 недели
**Уровень:** Промежуточный (нужен опыт Python)

### Структура по неделям

**Неделя 1: Use cases, жизненный цикл проекта и pre-training (~5 ч 44 мин)**
Видеолекции:
- Generative AI & LLMs — обзор
- LLM use cases and tasks
- Text generation before transformers
- **Transformers architecture** — детальный разбор
- Generating text with transformers
- Prompting and prompt engineering
- Generative configuration (температура, top-k, top-p)
- Generative AI project lifecycle
- Pre-training large language models
- Computational challenges of training LLMs
- **Scaling laws and compute-optimal models** (Chinchilla scaling laws)
- Pre-training for domain adaptation

**Лаб 1:** Dialogue Summarization (120 мин) — подготовка данных через PyTorch, few-shot inference

**Неделя 2: Fine-tuning и оценка LLM (~4 ч 39 мин)**
Видеолекции:
- Instruction fine-tuning
- Fine-tuning on a single task
- Multi-task instruction fine-tuning
- Model evaluation (метрики)
- Benchmarks
- Catastrophic forgetting и как его предотвратить
- **Parameter Efficient Fine Tuning (PEFT)**
- LoRA (Low-Rank Adaptation)
- Soft prompts
- Scaling instruct models

**Лаб 2:** PEFT для dialogue summarization (120 мин)

**Неделя 3: RLHF и LLM-powered приложения (~6 ч)**
- **RLHF** (Reinforcement Learning from Human Feedback)
- Обучение reward model с помощью человеческой обратной связи
- Chain-of-thought prompting для улучшения рассуждений
- Knowledge cut-off и retrieval augmentation (RAG)
- Интеграция LLM с внешними кодовыми интерпретаторами
- Подключение внешних источников данных

**Лаб 3:** FLAN-T5 + reinforcement learning для генерации позитивных саммари (120 мин)

### Уникальная педагогика
- Глубокий технический разбор трансформеров, доступный для промежуточного уровня
- 3 практические лабораторные на AWS SageMaker
- Ссылки на оригинальные научные статьи + саммари
- Баланс теории (архитектура) и практики (лабораторные)

### Как объясняют трансформеры неспециалистам
- Последовательный путь: N-граммы -> RNN -> Transformers
- Объяснение self-attention как механизма работы с контекстом
- Визуализации и примеры генерации текста

### Что можно позаимствовать
- Структуру "pre-training -> fine-tuning -> RLHF -> deployment" как карту LLM-проекта
- Объяснение scaling laws (Chinchilla) — важная концепция для понимания экономики LLM
- Лабораторные работы как шаблон для практических заданий
- RAG и chain-of-thought как прикладные техники

## 4. fast.ai — "Practical Deep Learning for Coders" (Jeremy Howard)

**Платформа:** fast.ai (бесплатно)
**URL:** https://course.fast.ai/
**Преподаватель:** Jeremy Howard
**Длительность:** 9 уроков (Part 1) + 17 уроков (Part 2), каждый ~90 мин
**Уровень:** Практический, от начинающего до продвинутого
**Инструменты:** PyTorch, fastai, Hugging Face Transformers, Gradio

### Part 1: 9 уроков

**Урок 1: Getting Started**
- End-to-end пример обучения и использования модели за 5 минут
- Создание моделей на собственных данных
- Введение в Jupyter Notebook и библиотеку fastai
- Ноутбук: "Is it a bird? Creating a model from your own data"

**Урок 2: Deployment**
- Дизайн собственного ML-проекта
- Создание собственного датасета
- Обучение модели
- Развертывание приложения на Hugging Face Space с Gradio

**Урок 3: Neural Net Foundations**
- Стохастический градиентный спуск (SGD)
- Линейные функции с нелинейными активациями
- Rectified Linear Function (ReLU)

**Урок 4: Natural Language (NLP)**
- Анализ естественного языка
- Hugging Face ecosystem, Transformers library
- Проект: классификация сходства фраз из патентов США

**Урок 5: From-scratch Model**
- Создание нейронной сети с нуля на Python и PyTorch
- Реализация training loop для оптимизации весов

**Урок 6: Random Forests**
- Как работают случайные леса
- Построение random forest с нуля
- Интерпретация моделей для понимания данных

**Урок 7: Collaborative Filtering**
- Рекомендательные системы
- Matrix completion через линейную алгебру
- Deep learning подход к коллаборативной фильтрации
- Embeddings, cross-entropy, softmax, label smoothing

**Урок 8: Convolutions (CNNs)**
- Embeddings в deep learning
- Свёрточные нейронные сети: механика работы
- Операции свёртки, pooling, dropout

**Бонус: Data Ethics**
- Ответственное применение AI
