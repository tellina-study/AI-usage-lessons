# Routines

## Daily

1. **Sync** ‚Äî run `sync-library` to pull Google Drive changes
2. **Catalog** ‚Äî run `catalog-docs` + `extract-links` to update index and ontology
3. **Tasks** ‚Äî `issue-from-change` creates/updates GitHub Issues for detected changes
4. **Content** ‚Äî `update-lecture`, `build-deck`, `diagram-refresh` as needed

## Weekly

1. **Impact check** ‚Äî run `impact-check` to find what changed and what it affects
2. **Issue triage** ‚Äî `issue-manager` agent reviews and prioritizes open issues
3. **Notes** ‚Äî update `notes/decisions.md` with any new decisions or findings
